# 기존 페이지 레이아웃 전환 가이드

## 적용 순서

1. 각 페이지에서 최상위 폭 제어 클래스를 `.dure-container`로 변경
2. 섹션 상하 여백을 `.dure-section`으로 변경
3. 반복 카드 목록을 `.dure-grid`로 변경
4. 텍스트+이미지 구성을 `.dure-split`으로 변경
5. CTA 영역을 `.dure-cta`로 변경
6. 남은 페이지 전용 CSS만 `assets/css/pages/`에 유지

## 권장 우선순위

1. 메인 페이지
2. 가족가게
3. 이웃가게
4. 안다미로 상세
5. 가족가게 신청
