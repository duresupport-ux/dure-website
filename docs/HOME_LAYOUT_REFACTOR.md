# 메인 페이지 Layout System 적용

## 적용된 공통 컴포넌트

- `dure-container`
- `dure-section`
- `dure-hero`
- `dure-section-head`
- `dure-cta`
- `dure-grid`
- `dure-card`
- `dure-button`

## 결과

메인 페이지의 공통 레이아웃 코드는 `layout.css`, 버튼·카드 코드는
`components.css`에서 관리합니다.

`home.css`에는 배경, 이미지 영역, 메인 전용 카드 배치와 같은 페이지 고유 스타일만 남겼습니다.

다른 페이지도 같은 원칙으로 순차 전환합니다.
