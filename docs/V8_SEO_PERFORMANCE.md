# DURE Website v8 — SEO & Performance

## SEO

- 페이지별 고유 title과 description
- canonical URL
- Open Graph 및 Twitter Card
- Organization / WebSite / WebPage / Service / Restaurant JSON-LD
- robots.txt
- sitemap.xml
- favicon 및 web manifest
- 내부 개발 페이지 noindex

## 성능

- PNG/JPG → WebP 변환
- 모든 이미지 width/height 지정
- 첫 콘텐츠 이미지 eager + fetchpriority high
- 나머지 이미지 lazy loading
- video preload=metadata
- iframe lazy loading
