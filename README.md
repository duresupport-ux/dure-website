# 두레 홈페이지 배포 파일

최종 기준 도메인: `https://dure.io`

## 페이지 구조

- `/` — 두레 메인
- `/family/` — 가족가게 안내
- `/neighbor/` — 이웃가게 안내
- `/apply/family/` — 가족가게 신청 안내
- `/stores/andamiro/` — 안다미로 스시 상세 페이지

## 배포 방법

Cloudflare Pages에 이 폴더의 **내용 전체**를 배포하면 됩니다.

- 프레임워크: 없음(None)
- 빌드 명령어: 비움
- 배포 디렉터리: 저장소 루트(`/`)
- 프로덕션 브랜치: `main`

## 현재 외부 연결

- 가족가게 최종 신청: Google Forms
- 이웃가게 신청 및 일부 참여가게 상세 콘텐츠: 기존 Notion 페이지
- 안다미로 매장 확인: 네이버 검색

## 파일 관리 원칙

HTML 내부의 Base64 이미지·영상은 `/assets/`로 분리했습니다. 이미지가 중복될 경우 하나의 파일을 공용으로 사용하며, 큰 이미지는 WebP로 최적화했습니다.
